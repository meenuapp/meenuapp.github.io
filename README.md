# MeenuApp Brand Portfolio
This repository contains the professional portfolio and privacy policies for MeenuApp / EmbroBees mobile games and applications.

## Project Structure
- `index.html`: The professional landing page (Portfolio).
- `word-connect-pro-privacy.html`: Dedicated privacy policy for "Word Connect Pro".
- `privacy-policy.md`: Generic privacy policy for all older MeenuApp games.
- `docs/`: Historical implementation plans and documentation.

## How to Update
1. To change the home page, edit `index.html`.
2. To update privacy policies, edit the respective `.html` or `.md` files.
3. Push changes to the `meenuapp.github.io` repository on GitHub to make them live.
